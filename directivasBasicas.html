Directivas Básicas

1. ¿Qué estoy aprendiendo?

En este apartado veremos el archivo de configuración global encargado de contener las directivas globales del servidor web Apache. Se encuentran dentro de las variables de entorno, situadas en /etc/apache2/envvars. Se basan en unas variables que recogen distintos valores, como pueden ser el nombre del servidor, rutas predefinidas de configuración, de los documentos... El objetivo de estas variables de entorno es construir "atajos" para no repetir código en distintos archivos, lo que sería un trabajo tedioso si realizamos alguna modificación. Con estas variables nos aseguramos que en un solo fichero cambiamos el valor en todos aquellos archivos donde hagamos referencia.

2. ¿Qué recursos estoy utilizando?

Disponemos de un material didáctico mediante diapositivas para el aprendizaje de este bloque, ademas de las horas de clase para aprender de forma práctica.

3. ¿Qué dificultades estoy encontrando?

Este apartado es aparentemente sencillo de entender y facilita el trabajo de configuración de un servidor.
