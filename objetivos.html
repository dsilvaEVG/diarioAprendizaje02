Objetivo del diario de aprendizaje

El objetivo del diario de aprendizaje es responder a tres cuestiones clave respecto a lo aprendido durante este bloque:

1. ¿Qué estoy aprendiendo?

2. ¿Qué recursos estoy utilizando?

3. ¿Qué dificultades estoy encontrando?

Para ello, vamos a dividir este bloque de GIT en varios apartados para ayudar a procesar el aprendizaje.
