Protocolo HTTP

1. ¿Qué estoy aprendiendo?

En la primera parte del módulo hemos aprendido cómo funciona el protocolo HTTP: Cómo se establece la conexión entre el cliente y el servidor, las funciones que ejercen ambos dentro de la conexión, módulos y lenguajes que soportan cada uno, la forma en que se comunican entre sí, etc.

También hemos visto el Modelo Vista Controlador, el certificado SSL (su proceso y utilidad dentro de la seguridad en las páginas web, diferencias entre HTTP vs HTTPS), los estados de la página (errores cliente, errores servidor, éxito…), entre otros aspectos.

2. ¿Qué recursos estoy utilizando?

Para esta unidad didáctica hemos usado el recurso de Classroom de la asignatura, sirviendo de base para las clases presenciales.

3. ¿Qué dificultades estoy encontrando?

En este apartado no existen dificultades, el contenido es claro y las diapositivas de clase aportan contenido gráfico que ayuda a la comprensión.
